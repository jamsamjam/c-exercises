# C Exercises

This repository contains C programming exercises from the CS-202 course.

## How to Compile

```sh
gcc -o name name.c
```

For a stricter standard-compliant compilation:
```sh
gcc -ansi -Wall -pedantic -o name name.c
gcc -std=c17 -Wall -pedantic -o name name.c
```

## How to Run

```sh
./name
```
